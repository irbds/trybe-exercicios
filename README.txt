# trybe-exercicios

#### Treino de logica de programação


--3°--
1-cria uma linha com n character
  1.1-se a coluna da linha atual for a n-1 cria um "*" se não cria um " "
2-pula uma linha e cria outra linha com n character
  2.1-se a coluna da linha atual for a n-2 ou maior cria um "*" se não cria um " "

--4°--

1-Recebe um numero de * para a base da piramide
2-Cria uma array apenas com os numeros impares para a estrutura da piramide
3-Pega o valor da base da piramide, divide por 2
4-Armazena o valor da divisao em uma variavel chamada ValorEspaco
6-Cria uma array chamada linhas para armazenar o valor de cada linha
5-Cria a primeira linha com ValorEspaco de espacos e adiciona um "*" no final
6-Pula para proxima linha
7-Pega o numero impar anterior ao numero da base e repete os passos ate o numero pego ser maior que a base
